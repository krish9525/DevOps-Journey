# Day 1 - Basic Shell Scripting

## Commands Learned:

- date → shows current date
- whoami → shows current user
- pwd → shows current directory
- ls -l → lists files
- df -h → disk usage
- free -h → memory usage
- touch → create file
- cat → read file

## What I learned:
- How to write a basic shell script
- How to make it executable
- How to run scripts in Linux
